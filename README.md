NumPy Assignment – Basics, Cleaning, and Analysis
📖 Overview
This project demonstrates NumPy fundamentals through three Python scripts:

numpy_basics.py → Core NumPy operations and array manipulations

data_cleaning_numpy.py → Handling missing values and cleaning numeric arrays

data_analysis_numpy.py → Performing simple statistical analysis

It is designed as a practice assignment to strengthen your understanding of NumPy and numerical computing in Python.
